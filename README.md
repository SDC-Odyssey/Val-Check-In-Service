# peter-waterBnB

Check In module for WaterBnB

This repository contains the 1 module -- the check-in module for WaterBnB

### Local development

First seed the db using,

```
$ npm run db-seed
```

To run the server,

```
$ npm run server-dev
```

To build the client,

```
$ npm run build

or

$ npm run react-dev
```

To run the tests,

```
$ npm test
```
